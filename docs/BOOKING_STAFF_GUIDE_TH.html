1. ภาพรวมระบบ

ระบบนัดหมาย TestD ช่วยให้ผู้ใช้บริการสามารถจองนัดหมายที่สาขาใดก็ได้ใน 4 สาขาของ SWING รองรับทั้งผู้ใช้ที่สมัครสมาชิกและผู้ใช้ทั่วไป (ไม่ต้องสมัครสมาชิก)

ฟีเจอร์หลัก

  • นัดหมายหลายบริการ — เลือกหลายบริการได้ในนัดเดียว
  • การจัดตารางตามความหนาแน่น — ช่วงเวลาแสดงระดับความแออัด (น้อย/ปานกลาง/มาก) เพื่อกระจายภาระงาน
  • ประมาณเวลารอ — คำนวณจากจำนวนที่ปรึกษา น้ำหนักบริการ และจำนวน Walk-in
  • รหัสอ้างอิง — ทุกนัดหมายได้รหัสเฉพาะ (เช่น SWG-A3K9X2) สำหรับเช็คอิน
  • อัปเดตแบบเรียลไทม์ — แดชบอร์ดเจ้าหน้าที่อัปเดตทันทีเมื่อมีการจองหรือเปลี่ยนแปลง
  • จองแบบ Guest — ไม่ต้องสมัครสมาชิก ใช้แค่อีเมล

2. ขั้นตอนการจองสำหรับผู้ใช้บริการ

นี่คือสิ่งที่ผู้ใช้บริการเห็นเมื่อเข้าหน้า /booking:

1 เลือกสาขา

เลือกจาก 4 สาขา แต่ละสาขาแสดงรูปภาพ คะแนน Google Maps ที่อยู่ และเวลาเปิดทำการ

2 เลือกบริการ

เลือกได้หลายบริการ (ตรวจ HIV, PrEP, PEP, STI ฯลฯ) ระบบแสดงสิทธิ์ค่าบริการอัตโนมัติ

3 เลือกวันและเวลา

ปฏิทิน + ช่วงเวลาแสดงความหนาแน่น: 🟢 น้อย, 🟡 ปานกลาง, 🔴 มาก

4 ยืนยันและรับรหัส

จองสำเร็จ → แสดงรหัสอ้างอิง ระบบแนะนำให้แคปหน้าจอไว้สำหรับเช็คอิน

💡 คำแนะนำอัจฉริยะ หากผู้ใช้เลือกช่วงเวลาที่แออัด ระบบจะแนะนำช่วงเวลาอื่นที่ว่างกว่าโดยอัตโนมัติ เพื่อกระจายภาระงานตลอดวัน

3. การจองแบบไม่ต้องสมัครสมาชิก (Guest Booking)

ผู้ใช้บริการไม่จำเป็นต้องสมัครสมาชิกก็จองได้ ต้องการแค่อีเมลเท่านั้น

วิธีการทำงาน:

  1. ผู้ใช้ทำตามขั้นตอนปกติ (เลือกสาขา → บริการ → วัน/เวลา)
  2. แทนการล็อกอิน ให้กรอกอีเมลติดต่อ
  3. ระบบสร้างนัดหมายอย่างปลอดภัยผ่านฟังก์ชันฝั่งเซิร์ฟเวอร์
  4. ผู้ใช้ได้รับรหัสอ้างอิงในหน้ายืนยันสำเร็จ
✅ การเชื่อมบัญชี หลังจองสำเร็จ ระบบจะแสดงข้อความ: "อยากจัดการนัดง่ายขึ้นไหม? สมัครสมาชิกเพื่อดู เลื่อน หรือยกเลิกนัดได้ทันที"
หากผู้ใช้สมัครด้วยอีเมลเดียวกัน ระบบจะเสนอให้เชื่อมนัดหมายเดิมกับบัญชีใหม่

4. รหัสอ้างอิง (Referral Code)

📌 รูปแบบ: SWG-XXXXXX
สร้างอัตโนมัติโดยระบบฐานข้อมูล ไม่ซ้ำกันในแต่ละนัดหมาย

สำหรับผู้ใช้บริการ:

  • แสดงอย่างชัดเจนในหน้ายืนยันการจอง
  • ข้อความที่แสดง: "กรุณาแคปหน้าจอนี้และแสดงให้เจ้าหน้าที่ลงทะเบียน"

สำหรับเจ้าหน้าที่:

  • เปิดหน้า Admin → การจอง
  • ใช้ช่องค้นหารหัสอ้างอิงเพื่อค้นหานัดหมายทันที
  • การค้นหานี้ข้ามตัวกรองวันที่ได้ — ใช้ได้แม้จะกำลังดูวันอื่นอยู่
  • ตรวจสอบตัวตนผู้ใช้ → อัปเดตสถานะเป็น มาถึงแล้ว
⚠️ สำคัญ ตรวจสอบให้แน่ใจว่ารหัสอ้างอิงตรงกับผู้ใช้บริการ ขอให้แสดงหน้าจอที่แคปไว้หรืออีเมลยืนยัน

5. แดชบอร์ดเจ้าหน้าที่

เข้าถึงผ่าน: Admin → แท็บการจอง (ต้องมีสิทธิ์ admin หรือ staff)

5.1 Bento Dashboard (ภาพรวมรายวัน)

หน้าหลักแสดงตัวเลขสรุปของวันนี้:

ตัวชี้วัดคำอธิบาย
ทั้งหมดจำนวนนัดหมายทั้งหมดของวัน
ใหม่ผู้ใช้บริการครั้งแรก
กลับมาใช้บริการผู้ใช้ที่เคยมาก่อนหน้านี้
ยกเลิกนัดหมายที่ถูกยกเลิกในวันนี้

แต่ละสาขาจะแสดงตัวบ่งชี้ภาระงานเพื่อให้เห็นว่าสาขาไหนยุ่งที่สุด

5.2 ไทม์ไลน์สาขา

  • คลิกการ์ดสาขา → เปิดไทม์ไลน์แนวตั้งของนัดหมายทั้งหมด
  • แต่ละนัดหมายแสดงเป็นแถบพร้อมป้ายสถานะและไอคอนบริการ
  • คลิกแถบใดก็ได้ → เปิดลิ้นชักรายละเอียดนัดหมาย

5.3 ลิ้นชักรายละเอียดนัดหมาย

ในลิ้นชักนี้ เจ้าหน้าที่สามารถ:

  • ดูรายละเอียดนัดหมายทั้งหมด (บริการ เวลา ข้อมูลผู้ใช้)
  • อัปเดตสถานะ (ยืนยัน → มาถึง → กำลังให้บริการ → เสร็จสิ้น)
  • มอบหมายเจ้าหน้าที่ดูแลนัดหมาย
  • เพิ่มบันทึกภายใน (เจ้าหน้าที่เห็นเท่านั้น)

5.4 การดำเนินการแบบกลุ่ม

  • เลือกหลายนัดหมายโดยใช้ช่องทำเครื่องหมาย
  • อัปเดตสถานะพร้อมกัน — เช่น ทำเครื่องหมาย 5 รายการเป็น "มาถึงแล้ว" ในครั้งเดียว
  • ส่งออก CSV — ดาวน์โหลดนัดหมายที่เลือกสำหรับรายงาน

6. การจัดการ Walk-in

สำหรับผู้ใช้บริการที่มาโดยไม่ได้จองล่วงหน้า:

1 สร้าง Walk-in

เจ้าหน้าที่กด "Walk-in ใหม่" ในแดชบอร์ด เลือกสาขาและเพิ่มบันทึก (ถ้ามี) ระบบสร้างนัดหมายแบบ source: 'walkin'

2 เริ่มบริการ

เมื่อที่ปรึกษาพร้อม กด "เริ่มบริการ" ระบบบันทึกเวลามาถึง

3 เสร็จสิ้นบริการ

หลังให้คำปรึกษาเสร็จ กด "เสร็จสิ้น" ระบบบันทึกเวลาเสร็จ

4 ผลต่อเวลารอ

Walk-in ที่ active จะเพิ่มค่าความกดดัน Walk-in ทำให้เวลารอโดยประมาณของผู้ใช้รายอื่นเพิ่มขึ้น

💡 ลงทะเบียนด่วน (Quick Register) แผง Admin ยังมีแท็บ "ลงทะเบียนด่วน" ที่เจ้าหน้าที่สามารถลงทะเบียนผู้มาใช้บริการ Walk-in ด้วยตนเอง หรือใช้การสแกน OCR บัตรประชาชน — ไม่ต้องใช้อุปกรณ์หรืออินเทอร์เน็ตของผู้ใช้บริการ

10. บัตรอ้างอิงด่วน

🏥 ขั้นตอนการทำงานประจำวันสำหรับเจ้าหน้าที่ลงทะเบียน
  1. เปิด Admin → การจอง ตอนเริ่มงานทุกวัน
  2. ตรวจสอบ Bento Dashboard ดูจำนวนนัดหมายวันนี้ของแต่ละสาขา
  3. เมื่อผู้ใช้บริการมาถึง → ถามรหัสอ้างอิงหรือหน้าจอที่แคปไว้
  4. ค้นหารหัสในแดชบอร์ด → ตรวจสอบรายละเอียด → ทำเครื่องหมายเป็น มาถึงแล้ว
  5. หากเป็น Walk-in → กด "Walk-in ใหม่" → กด "เริ่มบริการ" เมื่อพร้อม
  6. หลังให้คำปรึกษา → ทำเครื่องหมายเป็น เสร็จสิ้น
  7. สิ้นสุดวัน → ตรวจสอบรายการ ไม่มา (No-show) และอัปเดตสถานะ
🔍 ค้นหานัดหมายอย่างรวดเร็ว
  • ค้นหาด้วยรหัสอ้างอิง: ใช้ช่องค้นหา — ข้ามตัวกรองวันที่ได้
  • ค้นหาตามวันที่: ใช้ปฏิทินไปยังวันที่ของนัดหมาย
  • ค้นหาตามสาขา: คลิกการ์ดสาขาเพื่อดูไทม์ไลน์

ช่วงเวลาแสดง "เต็ม"ตรวจสอบว่ามีนัดที่ถูกยกเลิกที่ทำให้ว่างหรือไม่ ปรับจำนวนที่ปรึกษาถ้าจำเป็น
Guest ต้องการจัดการนัดหมายแนะนำให้สมัครสมาชิก — ระบบสามารถเชื่อมนัดหมายเดิมได้
