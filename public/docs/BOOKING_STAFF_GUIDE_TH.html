1. ภาพรวมระบบ

ระบบนัดหมาย TestD ช่วยให้ผู้ใช้บริการจองนัดหมายที่สาขาใดก็ได้ใน 4 สาขาของ SWING รองรับผู้ใช้สมาชิก, ผู้ใช้ Guest (ใช้แค่อีเมล) และ Walk-in

ฟีเจอร์หลัก

  • นัดหมายหลายบริการ — เลือกหลายบริการได้ในนัดเดียว
  • ตารางตามความหนาแน่น — ช่วงเวลาแสดง 🟢 ว่าง / 🟡 ปานกลาง / 🔴 หนาแน่น
  • ระบบ Atomic — ป้องกัน double booking ด้วย database lock V2
  • จองวันนี้ได้ — แสดงเฉพาะ slot ที่ยังไม่ผ่าน V2
  • ประมาณเวลารอ — คำนวณจากจำนวนที่ปรึกษา, น้ำหนักบริการ, Walk-in
  • รหัสอ้างอิง — ทุกนัดได้รหัส SWG-XXXXXX
  • อัปเดตเรียลไทม์ — แดชบอร์ดอัปเดตทันที
  • Guest booking — ใช้แค่อีเมล + ได้ลิงก์ token จัดการนัด V2
  • 6 ภาษา — TH, EN, KH, LA, VI, MY V2
  • Blackout — ปิดวัน/เวลาสำหรับวันหยุดได้ V2
  • แบบประเมินความเสี่ยง — แนะนำบริการอัตโนมัติ

2. ขั้นตอนการจอง 4 ขั้น

ผู้ใช้เข้าหน้า /booking จะเห็น progress bar 4 ขั้นตอน:

ขั้นตอนที่ 1: เลือกสาขา

การ์ดสาขาแสดง: รูปภาพ, คะแนน Google Maps, ที่อยู่, วัน/เวลาเปิดทำการ, ลิงก์ไป Google Maps

📱 ภาพหน้าจอ: หน้าเลือกสาขา
การ์ด 4 สาขา พร้อมรูป คะแนน Google และเวลาเปิด
หน้าจอเลือกสาขา — ผู้ใช้เห็นการ์ดสาขาพร้อมรายละเอียด

ขั้นตอนที่ 2: เลือกบริการ

เลือกได้หลายบริการ พร้อมป้ายสิทธิ์ (🇹🇭 ฟรี คนไทย NHSO, 🌍 ฟรี กองทุนโลก, 💰 เสียค่าบริการ) มีปุ่ม "ไม่แน่ใจ? ทำแบบประเมินความเสี่ยง" เพื่อแนะนำบริการอัตโนมัติ

📱 ภาพหน้าจอ: หน้าเลือกบริการ
รายการบริการพร้อม checkbox และป้ายสิทธิ์ฟรี + ปุ่มแบบประเมิน
หน้าจอเลือกบริการ — ติ๊กเลือกได้หลายรายการ

ขั้นตอนที่ 3: เลือกวันและเวลา

แถบวันที่แนวนอนเลื่อนได้ (30 วัน) แสดงเฉพาะวันที่สาขาเปิด วันที่ Blackout จะขีดฆ่า ปุ่ม "วันนี้" กะพริบ V2

ช่วงเวลาใช้ DensityTimeSelector แสดงช่วง 2 ชม. พร้อมสี density และเวลารอโดยประมาณ

📱 ภาพหน้าจอ: หน้าเลือกวัน/เวลา
ปฏิทินแนวนอน + ช่วงเวลาพร้อมสี density
หน้าจอเลือกวัน/เวลา
💡 คำแนะนำอัจฉริยะ หากผู้ใช้เลือกช่วงเวลาที่แออัด ระบบจะแนะนำช่วงเวลาอื่นที่ว่างกว่า

ขั้นตอนที่ 4: ยืนยันและรับรหัส

สรุปรายละเอียด + ช่องกรอกบันทึก (ถ้ามี) Guest กรอกอีเมลที่ขั้นตอนนี้ จองสำเร็จ → แสดงรหัส + ปุ่มคัดลอก + แนะนำให้แคปหน้าจอ

📱 ภาพหน้าจอ: หน้ายืนยันสำเร็จ
รหัสอ้างอิง SWG-XXXXXX + สรุปนัดหมาย + ปุ่มคัดลอก
หน้าจอยืนยัน — ผู้ใช้ควรแคปหน้าจอนี้

3. การจองแบบ Guest + Token Access ใหม่

ไม่ต้องสมัครสมาชิก ต้องการแค่อีเมล

วิธีการทำงาน:

  1. ผู้ใช้ทำตาม 4 ขั้นตอนปกติโดยไม่ล็อกอิน
  2. ขั้นตอนยืนยัน กรอกอีเมลติดต่อ
  3. ระบบเรียก create_anonymous_appointment (server-side, bypass RLS)
  4. ได้รับรหัสอ้างอิงทันที
  5. ระบบสร้างGuest Access Token — ลิงก์จัดการนัดไม่ต้องสมัคร V2
  6. ส่งอีเมลยืนยันพร้อมรหัส + ลิงก์
  7. นัดหมายถูกบันทึกใน localStorage ด้วย
✅ Guest Access Token V2 Guest ได้รับลิงก์พิเศษ /guest-appointments?token=... เพื่อดูและจัดการนัดหมาย ไม่ต้องสมัครสมาชิก Token มีกำหนดหมดอายุ
✅ การเชื่อมบัญชี หลังจองสำเร็จ แสดง: "อยากจัดการนัดง่ายขึ้นไหม? สมัครสมาชิกเพื่อดู เลื่อน หรือยกเลิกนัดได้ทันที" หากสมัครด้วยอีเมลเดียวกัน ระบบเชื่อมนัดเดิมให้อัตโนมัติ

4. รหัสอ้างอิง (Referral Code)

📌 รูปแบบ: SWG-XXXXXX
สร้างอัตโนมัติโดยระบบ ไม่ซ้ำกัน ทั้ง Guest และสมาชิกได้รับรหัส

สำหรับผู้ใช้บริการ:

  • แสดงในหน้ายืนยันพร้อมปุ่มคัดลอก
  • ข้อความ: "กรุณาแคปหน้าจอนี้และแสดงให้เจ้าหน้าที่ลงทะเบียน"
  • รหัสอยู่ในอีเมลยืนยันด้วย

สำหรับเจ้าหน้าที่:

  • เปิด Admin → การจอง
  • ใช้ช่องค้นหารหัส — ข้ามตัวกรองวันที่ได้
  • ตรวจสอบตัวตน → อัปเดตสถานะ
⚠️ สำคัญ ตรวจสอบให้แน่ใจว่ารหัสตรงกับผู้ใช้ ขอให้แสดงหน้าจอที่แคปหรืออีเมลยืนยัน

6. การจัดการ Walk-in

อยู่ใน Bento Dashboard เลย ไม่ต้องเปลี่ยนหน้า

1 สร้าง Walk-in

เลือกสาขา → กด "+ Walk-in" ระบบสร้างนัดแบบ source: walkin สถานะ: waiting พร้อมรหัสอ้างอิง

2 เริ่มบริการ

เมื่อพร้อม กด "เริ่ม" (▶️) สถานะเปลี่ยนเป็น in_progress

3 เสร็จสิ้น

กด "เสร็จ" (✅) สถานะเปลี่ยนเป็น completed

4 ผลต่อเวลารอ

Walk-in ที่ active เพิ่มค่า walk-in pressure → เวลารอของผู้อื่นเพิ่มขึ้น

🖥️ ภาพหน้าจอ: Walk-in ในแดชบอร์ด
ปุ่ม "+ Walk-in" + รายการ active walk-ins + ปุ่มเริ่ม/เสร็จ

12. บัตรอ้างอิงด่วน

🏥 ขั้นตอนการทำงานประจำวัน
  1. เปิด Admin → การจอง ตอนเริ่มงาน
  2. ตรวจ Bento Dashboard — ดูจำนวนนัด, ระดับความหนาแน่น, ภาระแต่ละสาขา
  3. ผู้ใช้มาถึง → ถามรหัสอ้างอิงหรือหน้าจอที่แคป
  4. ค้นหารหัส → ตรวจสอบ → ทำเครื่องหมาย ยืนยัน หรือ กำลังรับบริการ
  5. Walk-in → เลือกสาขา → กด "+ Walk-in" → เริ่มเมื่อพร้อม
  6. หลังให้คำปรึกษา → เสร็จสิ้น + เพิ่ม staff notes
  7. สิ้นสุดวัน → ตรวจ ไม่มา (No-show) (ระบบ cron ทำเครื่องหมายอัตโนมัติ)
🔍 ค้นหานัดหมายอย่างรวดเร็ว
  • ด้วยรหัสอ้างอิง: ช่องค้นหา — ข้ามวันที่
  • ตามวันที่: ปฏิทิน
  • ตามสาขา: คลิกการ์ดสาขา
  • ตามเวลา: คลิกชั่วโมงพีค
